### 6. Find an Euclidian distance between (2, 3) and (10, 8)

```python

punto2 = (10, 8)

# CÃ¡lculo de la distancia euclidiana
distancia = ((punto2[0] - punto1[0])**2 + (punto2[1] - punto1[1])**2) ** 0.5
```
